# EZGO Landing Page

SEO-optimized marketing website for EZGO platform.

## Features

- ✅ **SEO Optimized**: Complete metadata, Open Graph, Twitter Cards
- ✅ **Structured Data**: JSON-LD for better search engine understanding
- ✅ **Sitemap**: Auto-generated sitemap.xml
- ✅ **Robots.txt**: Proper crawling instructions
- ✅ **Performance**: Next.js 15 with App Router
- ✅ **Responsive**: Mobile-first design
- ✅ **Accessibility**: WCAG compliant

## Running Locally

```bash
npm run dev
```

Runs on http://localhost:3002

## SEO Features

### Metadata
- Title, description, keywords
- Open Graph tags for social sharing
- Twitter Card tags
- Canonical URLs
- Language alternates (en, fr, ar)

### Structured Data (JSON-LD)
- Organization schema
- Contact information
- Social media links

### Performance
- Server-side rendering
- Optimized images
- Fast page loads

## Deployment

```bash
npm run build
npm start
```

Deploy to Vercel, Netlify, or any hosting platform that supports Next.js.
